# 麻将游戏皮肤制作完整教程

## 快速开始

### 最简单的方式：改色

只需要修改牌面的颜色，无需绘制完整图片：

1. **下载默认皮肤**：复制 `skin/default/` 文件夹
2. **改色**：
   - 牌面背景色：所有图片的背景色统一修改
   - 文字颜色：万/条/筒数字的颜色
   - 边框颜色：牌面边框的颜色
3. **重命名文件夹**：改成你想要的名称
4. **放入 skin/ 目录**：重启游戏即可

这种方式适合新手，无需专业技能。

## 进阶方式：完整定制

### 1. 准备工具

**推荐工具（按简单度排序）:**
- **Canva** (在线，免费，适合新手）
- **醒图** (手机APP，免费，适合快速制作)
- **GIMP** (免费，功能强大)
- **Photoshop** (专业，有免费替代品)

### 2. 下载模板

下载 `skin/template/` 文件夹中的模板文件：
- `table_bg_template.png` (1280×720)
- `mahjong_back_template.png` (80×100)
- `mahjong_front_template/` (所有牌面模板)
- `button_template/` (所有按钮模板)

### 3. 编辑步骤

#### 步骤1：编辑牌桌背景

打开 `table_bg_template.png`:
- 推荐尺寸：1280×720 像素
- 格式：PNG (支持透明)
- 颜色方案：
  - 国风：红木色+金色装饰
  - 日式：原木色+淡雅
  - 卡通：明亮多色

#### 步骤2：编辑麻将牌

打开 `mahjong_front_template/` 中的牌面文件：

**万子牌 (1wan.png ~ 9wan.png):**
- 背景：统一色系
- 数字：1-9，红色/黑色
- 装饰：可添加小元素

**条子牌 (1tiao.png ~ 9tiao.png):**
- 鸟/竹节图案
- 背景色与万子一致

**筒子牌 (1tong.png ~ 9tong.png):**
- 圆形图案
- 点数1-9

**风牌 (dong_feng.png, nan_feng.png, xi_feng.png, bei_feng.png):**
- 东/南/西/北字样
- 可用不同颜色区分

**箭牌 (zhong.png, fa.png, bai.png):**
- 中/发/白
- 中为红色，其他可不同色

**花牌 (chun.png ~ ju.png):**
- 春/夏/秋/冬/梅/兰/竹/菊
- 可用花卉图案

#### 步骤3：编辑麻将背面

打开 `mahjong_back_template.png`:
- 统一纹理或颜色
- 可添加小图案

#### 步骤4：编辑按钮

打开 `button_template/` 中的按钮文件：
- 吃/碰/杠/胡/过按钮
- 尺寸：60×40
- 风格：与牌桌一致

### 4. 保存和导出

1. 创建新文件夹：`skin/my_skin/`
2. 按相同结构保存所有图片
3. 确保命名与模板完全一致
4. 导出为PNG格式

### 5. 测试预览

在游戏中打开「皮肤中心」：
1. 点击「制作皮肤」下载模板
2. 编辑图片后放入 `skin/` 目录
3. 游戏会自动识别新皮肤
4. 选择预览效果

## 常见问题

**Q: 需要做所有144张牌吗？**

A: 不需要！可以：
1. 只改颜色（推荐新手）
2. 做几张代表性的牌，复制到其他
3. 使用批量处理工具

**Q: 图片格式有什么要求？**

A: 
- 格式：PNG (支持透明)
- 尺寸：见模板说明
- 文件名：必须与模板一致

**Q: 如何在游戏中测试？**

A: 
1. 将皮肤文件夹放入 `skin/` 目录
2. 重启游戏或刷新页面
3. 在「皮肤中心」中选择
4. 实时预览效果

**Q: 安卓端如何使用？**

A:
1. 用文件管理器找到游戏目录
2. 进入 `skin/` 文件夹
3. 放入你的皮肤文件夹
4. 重启游戏

**Q: 可以分享皮肤吗？**

A: 
1. 打包 `skin/` 中的你的皮肤文件夹
2. 分享ZIP文件
3. 其他人解压到 `skin/` 目录即可

## 高级技巧

### 1. 使用图层模板

下载PSD或GIMP分层模板：
- 底色层
- 文字层
- 装饰层
- 可单独修改每层

### 2. 批量处理

用脚本批量改色：
```bash
# ImageMagick 示例
mogrify -fill "#ff0000" -opaque white *.png
```

### 3. 主题一致性

确保以下元素颜色协调：
- 牌桌背景
- 牌面背景
- 牌面文字
- 按钮样式
