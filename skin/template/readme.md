# 麻将游戏皮肤制作模板说明

## 皮肤资源规范

### 图片格式要求
- **优先格式**：PNG（支持透明背景）
- **备选格式**：JPG（不透明背景）
- **禁止格式**：WebP（安卓8.0部分机型不兼容）

### 尺寸规范

| 资源类型 | 尺寸（像素） | 说明 |
|---------|-------------|------|
| 牌桌背景 | 1280×720 | 横屏核心尺寸，自动缩放 |
| 麻将正面 | 80×100 | 单张麻将牌 |
| 麻将背面 | 80×100 | 与正面尺寸一致 |
| 操作按钮 | 60×40 | 吃/碰/杠/胡/过按钮 |

### 命名规范

#### 麻将正面文件命名（必须严格遵循）

**万子牌（wan）：**
- 1wan.png ~ 9wan.png

- 1tiao.png ~ 9tiao.png

- nan_feng.png（南风）

**箭牌（jian）：**

**花牌（hua）：**
- chun.png（春）

## 制作步骤

### 1. 下载模板
下载 `skin/template/` 文件夹中的所有模板文件

### 2. 编辑图片
使用图片编辑工具（PS/醒图/Canva等）编辑模板

### 3. 导出图片
按照命名规范导出PNG图片

### 4. 创建皮肤文件夹
在 `skin/` 目录下创建你的皮肤文件夹，例如 `skin/my_skin/`

### 5. 放入资源
将编辑好的图片放入对应文件夹：
```
skin/my_skin/
├── table_bg.png
├── mahjong_back.png
├── mahjong_front/
│   ├── 1wan.png
│   ├── 2wan.png
│   └── ...（所有牌面图片）
└── button/
    ├── chi_btn.png
    ├── peng_btn.png
    └── ...（所有按钮图片）
```

## 快速换色方案

如果觉得144张麻将牌太繁琐，可以只修改以下核心元素：

1. **牌面底色**：统一修改所有牌面的背景色
2. **文字颜色**：修改牌面数字/文字的颜色
3. **边框样式**：修改牌面边框的粗细/颜色

游戏会自动应用这些基础样式到所有牌面。

## 注意事项

1. 所有图片命名必须与模板完全一致（包括大小写）
2. 图片尺寸尽量保持一致，避免拉伸变形
3. 透明背景请使用PNG格式
4. 建议先制作一张测试，确认效果后再批量制作

## 安卓端导入

1. 将皮肤文件夹打包成ZIP
2. 在游戏「皮肤中心」选择「导入皮肤」
3. 选择ZIP文件，系统会自动解压并校验
4. 校验通过后皮肤会出现在皮肤列表中
